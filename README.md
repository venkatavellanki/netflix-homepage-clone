# Netflix Homepage Clone

## Project Overview
A responsive Netflix homepage clone built using HTML, CSS, and Bootstrap. This project is designed to help understand real-world UI layouts, Flexbox/Grid alignment, and responsive frontend development.

## Features
- Navbar with logo, menu items, and profile icon
- Hero banner with featured movie
- Multiple horizontal movie rows (Trending, Popular, Top Rated, Genres)
- Movie hover effects
- Fully responsive layout
- Footer with copyright

## Technologies Used
HTML5, CSS3, Bootstrap 5, JavaScript (optional)
